Odoo integration with Git

Integrate your development directly with Odoo Project management

Track code changes with your tasks

Have a better traceability over changes made for a specific task

Commit message making reference to a task get append to the internal message thread. Allowing you to keep track of every commit referencing the task. Providing you with a history of changes made by developers.

Get notified of changes

Follow a task to get informed of code changes

People may choose to follow a task to receive notifications when developers are making modification to the code directly.

Working with Github and Gitlab

Easy to integrate with Github, Gitlab and other git repositories providing web hook

In Gitlab you simply need to add a new web hook in the "System Hooks" configuration.
